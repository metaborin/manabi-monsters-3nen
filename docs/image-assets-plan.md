## 画像仕様

- 形式: PNG
- 1024px版: 1024 x 1024
- 512px版: 512 x 512
- 背景: 透明PNG
- 画像内の文字: なし
- スタイル: 小学3年生向けの、かわいい2D教育ゲーム風
- 雰囲気: 明るい、安心感がある、集めたくなる、低学年にも親しみやすい
- 構図: カード、図鑑、結果画面、まなび島に配置しやすい中央構図

## 背景処理

GPT Imageで単色クロマキー背景の画像を生成したあと、ローカルで背景色を透明化した。透明化後にPillowで1024px版と512px版へリサイズし、すべてRGBA PNGとして保存した。

確認結果:

- 10ファイルすべてPNGとして読み込み可能
- 10ファイルすべてアルファチャンネルあり
- 1024px版はすべて1024 x 1024
- 512px版はすべて512 x 512
- 画像端の不透明ピクセルは0で、カード上に背景色の四角が出にくい状態

## 生成方針

### カズピョン

青系のうさぎ型モンスター。そろばん風の数え道具と星を持つ、元気で明るい算数担当キャラクター。画像内に文字や数字は入れない。

### コトバード

緑系の小鳥型モンスター。本、葉っぱ、鉛筆の雰囲気を持つ、やさしく賢そうな国語担当キャラクター。画像内に文字は入れない。

### ジッケンコ

黄色系のひよこ型研究者モンスター。ゴーグル、虫めがね、フラスコなどで観察と実験の雰囲気を出す。画像内に文字は入れない。

### チズモン

亀またはカメレオン風の社会担当モンスター。地図、町、方位磁針、リュックの雰囲気を持ち、やさしく案内してくれる印象にする。画像内に文字は入れない。

### ハロリン

ピンクと水色を基調に、虹、星、カード、風船の雰囲気を持つ英語担当モンスター。英語のあいさつや色の学習に合う明るさにする。画像内に文字は入れない。

## 今後アプリに組み込む場合の方針

既存のemoji表示をすぐ削除せず、段階的に画像表示へ置き換えるのが安全。

1. `src/data/monsters.ts` の各モンスターに画像パスを追加する。
2. 既存の `emoji` はフォールバック用として残す。
3. public配下の画像URLは `import.meta.env.BASE_URL` を使って解決する。
4. 図鑑、結果画面、まなび島の順に画像表示へ置き換える。
5. 画像読み込みに失敗した場合はemoji表示に戻す。
6. 375px程度のスマホ幅で、横スクロールやカード崩れがないか確認する。

アプリ本体へ組み込む候補箇所:

- `src/data/monsters.ts`
- `src/types/game.ts`
- `src/components/MonsterDex.tsx`
- `src/components/ResultScreen.tsx`
- `src/components/IslandScreen.tsx`
- `src/styles.css`
