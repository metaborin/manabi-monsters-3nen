# モンスター画像 品質チェック

実施日: 2026-07-04

対象フォルダ: `public/assets/monsters/`

## チェック結果

全PNGはRGBA形式で、アルファチャンネルあり。四隅と外周は透明で、黄緑・ピンクの不透明なベタ背景は検出されなかった。

初回確認後、プレビュー時の誤認を減らすため、全画像に最小限のクリーンアップを実施した。

- 透明ピクセルのRGB値を `0,0,0` に統一
- 低アルファのクロマキー色の縁残りを透明化
- モンスター本体の不透明部分、表情、色、構図は変更なし

## 機械検査の確認項目

- 1024px版はすべて `1024x1024`
- 512px版はすべて `512x512`
- PNGモードはすべて `RGBA`
- 四隅のアルファ値はすべて `0`
- 外周の可視ピクセルはすべて `0`
- 可視状態の黄緑・ピンク系クロマキー残りはすべて `0`
- 完全透明ピクセルに残るRGB値はすべて `0`

## 組み込み時のメモ

通常のカード、図鑑、リザルト、島表示には512px版を使うのが軽量。拡大表示や高解像度端末で大きく見せる画面を作る場合は1024px版を候補にする。

既存のモンスターID、クエストID、保存キーは変更せず、`src/data/monsters.ts` に画像パス用の任意フィールドを追加して、既存の絵文字はフォールバックとして残すのが安全。
